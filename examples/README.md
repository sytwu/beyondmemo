# BeyondMemo Examples

This directory contains example in-the-wild data for demonstrating BeyondMemo's explainability features.

## Datasets

### FI-London
Sample images from the FI-London dataset for testing and demonstration.

**Structure:**
```
FI-London/
├── Building_Attribute.json    # Metadata with attributes
└── Image/                      # Image files
    ├── 001_1.jpg
    ├── 009_9.jpg
    └── ...
```

### MapYourCity
Sample images from the MapYourCity dataset for testing and demonstration.

**Structure:**
```
MapYourCity/
├── 4cfqi7zewa/
│   ├── lat
│   ├── lng
│   ├── street.jpg
│   └── year_label
├── 4q6hleeitk/
└── ...
```

## Usage

Use these examples with the `playground.ipynb` notebook to:
- Test model inference on in-the-wild data
- Visualize explainability results
- Demonstrate BeyondMemo's capabilities

See the main repository README for more information.
